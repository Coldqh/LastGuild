# LastGuild v0.8.3 — Responsive UI Overhaul

## Main goals
- reduce interface density on phones;
- make laptop and desktop layout breathe more;
- add fast mobile navigation;
- improve topbar and resource readability;
- preserve existing systems while improving responsiveness.

## Added
- bottom mobile navigation bar for key sections;
- responsive topbar with adaptive time controls and resource pills;
- improved sidebar brand area with dark logo;
- PWA/browser icon links refreshed to dark v2 branding;
- safe-area support for iPhone home indicator;
- mobile-first modal behavior with slide-up full-width sheets.

## Improved
- layout spacing, card rounding and cross-device padding;
- map, archive, expedition and living-world layouts on tablet/mobile;
- character cards on phones;
- sidebar group readability;
- large-screen content width.

## Result
- cleaner phone UX;
- less overloaded laptop layout;
- better adaptation from ~360px mobile screens up to large desktop widths.
