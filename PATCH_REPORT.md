# PATCH REPORT — THE LAST GUILD v0.3

## Цель

Сделать персонажей и открытия постоянной частью истории кампании, а не временными числами внутри экспедиции.

## Добавлено

### Character simulation

- `CharacterCareerStage`;
- опыт и повышение уровня;
- профильное развитие навыков;
- расширенная память;
- записи травм с тяжестью, сроком лечения и необратимостью;
- старение и уход на покой;
- изменение отношений после успехов, провалов и решений лидера.

### Expedition debrief

- обязательный экран разбора после возвращения;
- несколько противоречивых отчётов;
- назначение автора открытия;
- четыре режима обработки находок;
- последствия для славы, лояльности, отношений и показателей гильдии.

### Discoveries and world consequences

- `DiscoveryRecord`;
- `WorldConsequence`;
- отложенная обработка последствий по календарю;
- изменение поселений, государств, опасности и репутации;
- отображение цепочек в архиве.

### Guild offices

- шесть должностей;
- один персонаж не может занимать несколько должностей;
- эффекты должностей встроены в расходы, лечение, слаженность, опыт, качество доказательств и публикации.

### UI

- расширенная карточка персонажа;
- отношения, карьера, XP, травмы и подробная память;
- новый архив с пятью разделами;
- экран разбора экспедиции;
- окно настроек;
- принудительное обновление приложения без удаления сохранения.

### Save system

- версия состояния повышена до `4`;
- новый ключ `last-guild-save-v4`;
- автоматическая миграция save v2/v3;
- нормализация старых травм и воспоминаний.
