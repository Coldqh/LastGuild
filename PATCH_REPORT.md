# THE LAST GUILD v0.5.1 — Core Loop & UX Audit

## Цель

Собрать существующие системы в понятный первый игровой цикл и снизить перегруз ранней игры без удаления глубины симуляции.

## Добавлено

- центр решений в штабе;
- цели первых 30 дней;
- пошаговое обучение из восьми действий;
- переход обучения к нужному экрану;
- управление временем «до события»;
- явная причина остановки времени;
- ручные слоты сохранений;
- экспорт и импорт кампании в JSON;
- резервная копия перед импортом и миграцией;
- восстановление резервного сохранения;
- переключатель конкурирующих гильдий;
- полное отключение чужих экспедиций, гонок и переманивания сотрудников;
- возможность вернуть конкурентов в текущую кампанию;
- настройки обучения и центра решений;
- стартовая запись о гибели прежней экспедиции;
- старый ветеран, переживший падение прежнего состава;
- версия приложения 0.5.1.

## Проверки

- чистый `npm ci`;
- `npm run build`;
- production preview — HTTP 200;
- SSR-генерация новой кампании;
- 180 стратегических дней без повторного появления отключённых конкурентов;
- импорт сохранения без конкурентов;
- ручной слот сохранения;
- миграция старого состояния сохранена.

## Ограничения

- обучение хранит прогресс локально в браузере;
- ручные слоты используют localStorage, IndexedDB планируется позже;
- баланс ранней игры улучшен интерфейсом и стартовой структурой, но требует живого плейтеста на нескольких полных кампаниях.
