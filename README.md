# THE LAST GUILD

Одиночный браузерный процедурный симулятор гильдии исследователей.

## v0.6.1 — Stability, Performance & Simulation Control

Патч укрепляет уже существующую игру вместо добавления нового слоя контента.

### Производительность

- тяжёлые разделы и модальные окна загружаются отдельными чанками;
- основной production-бандл уменьшен примерно с 504 кБ до 381 кБ;
- старые записи хроники, завершённые экспедиции и исторические данные ограничиваются безопасными лимитами;
- промотка времени останавливается сразу при появлении обязательного решения;
- добавлена единая функция «до следующего события».

### Управление симуляцией

В настройках можно отдельно отключить:

- войны;
- мировые кризисы;
- экономический упадок;
- смену границ;
- разрушение городов;
- старение;
- смерть от старости;
- отделение филиалов;
- переманивание сотрудников;
- конкурирующие гильдии.

Также доступны компактные карточки и настраиваемый лимит хроники.

### Баланс живого мира

- торговые пути больше не разрушаются неизбежно каждый месяц;
- безопасность маршрутов стремится к состоянию региона и может восстанавливаться;
- рост населения ограничивается вместимостью поселения;
- кризисы развиваются медленнее и учитывают устойчивость государства;
- город становится руинами только после настоящего тяжёлого коллапса;
- многолетние тесты больше не уничтожают большую часть мира за первое десятилетие.

### Инструменты разработки

Скрытая dev-панель позволяет:

- добавить ресурсы;
- открыть карту;
- вылечить сотрудников;
- завершить зависшие экспедиции;
- создать кризис или войну;
- провести аудит мира на 1, 10, 50 или 100 лет;
- скачать технический журнал состояния.

### Сохранения

Формат остаётся `save v7`. Патч не сбрасывает существующие кампании.

## Запуск

```bash
npm install
npm run dev
```

Проверка production-сборки:

```bash
npm run build
```
