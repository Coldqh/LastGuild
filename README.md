# THE LAST GUILD

Одиночный браузерный процедурный симулятор гильдии исследователей.

## v0.7.2 — Expedition UX Rework

### Новая структура экспедиций

В боковой панели разделены:

- **Контракты** — выбор цели и подготовка;
- **Активные походы** — управление отрядами в поле;
- **Архив** — законченные истории и знания.

### Пошаговая подготовка

Новый мандат проводится через шесть экранов:

1. цель;
2. команда;
3. маршрут;
4. снабжение;
5. приказы;
6. проверка.

Игра показывает недостающие профессии, конфликты, слабое здоровье, неизвестность маршрута, главную угрозу и конкретный способ усилить отряд.

### Активная экспедиция

Отдельный командный экран содержит:

- положение на маршруте;
- состояние связи;
- состав и здоровье;
- припасы и мораль;
- сгруппированный журнал;
- важные бои, травмы, смерти и открытия.

### Шаблоны

Есть готовые профили для разведки, руин, спасения и дальних походов. Собственный набор припасов и приказов можно сохранить локально.

### Возвращение

Основные итоги и решения показываются сразу. Подробные отчёты и списки убраны под раскрываемый блок.

## Запуск

```bash
npm install
npm run dev
```

Проверка production-сборки:

```bash
npm run build
```
