# THE LAST GUILD

Одиночный браузерный процедурный симулятор гильдии исследователей.

## v0.7 — Academy, Generations & Guild Politics

### Штаб

Штаб теперь является отдельным центром управления с вкладками:

- обзор;
- наём;
- помещения;
- должности;
- академия;
- совет.

Свободные кандидаты больше не смешиваются с действующим составом. Нанимать людей можно через `Штаб → Наём`.

### Академия

- восемь учебных программ;
- набор учеников из процедурного рынка людей;
- личные наставники;
- месяцы обучения;
- экзамены;
- выпуск в постоянный состав;
- развитие академии;
- школы и доктрины легендарных наставников.

### Поколения и политика

- поколения гильдии;
- совет;
- голосования;
- внутренние фракции;
- изменяемый устав;
- памятные объекты;
- институциональная память.

### Запуск

```bash
npm install
npm run dev
```

Production:

```bash
npm run build
```

Сохранение: `save v8`, старые кампании v2–v7 мигрируют автоматически.
