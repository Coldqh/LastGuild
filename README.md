# THE LAST GUILD

Одиночный браузерный процедурный симулятор гильдии исследователей.

## Первый патч

Рабочий вертикальный срез уже содержит:

- детерминированную генерацию мира по seed;
- гекс-карту 36×24;
- биомы, государства, столицы, поселения и границы;
- руины, подземелья, логова и популяции монстров;
- туман неизвестности и уровни изученности карты;
- 28 процедурных персонажей с навыками, чертами, страхами, амбициями и памятью;
- стартовый штат, рынок кандидатов, найм и расторжение контрактов;
- штаб, помещения, состояние здания, долг, жалование и месячные расходы;
- доску контрактов;
- создание экспедиции, подбор отряда, лидера, риска и припасов;
- поиск маршрута по карте;
- посуточную симуляцию похода, события, бой, болезни, открытия и возвращение;
- развитие репутации, знаний и персонажей;
- архив и хронику;
- localStorage-сохранение;
- GitHub Pages workflow.

## Запуск

```bash
npm install
npm run dev
```

Проверка production-сборки:

```bash
npm run build
```

## Ограничения v0.1

- бой пока рассчитывается как экспедиционное событие, клеточный viewer будет отдельным патчем;
- мировая политика пока генерируется, но не обновляется ежемесячно;
- подземелья существуют как исторические объекты без отдельных локальных схем;
- сохранение пока одно и работает через localStorage.
