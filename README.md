# THE LAST GUILD

Одиночный браузерный процедурный симулятор гильдии исследователей.

## v0.6 — Living World

Мир продолжает существовать без игрока:

- города производят и потребляют товары;
- торговые пути получают доход, безопасность и могут разрушаться;
- население растёт, беднеет, бунтует и покидает поселения;
- государства начинают войны из-за границ, маршрутов, ресурсов и документов;
- фронты меняют владельцев поселений;
- погибшие города превращаются в новые руины;
- открытия гильдии проверяются, публикуются, оспариваются и используются;
- новые маршруты меняют торговую сеть;
- каждый год сохраняется историческая карта мира.

### Экран «Живой мир»

Содержит:

- региональную экономику;
- торговые маршруты;
- рост и упадок городов;
- текущие и завершённые войны;
- распространение открытий;
- мировую хронику;
- исторические срезы карты.

### Настройки мира

При создании кампании можно изменить:

- скорость истории;
- частоту войн;
- жёсткость экономики;
- рост городов;
- катастрофы;
- силу влияния открытий.

### Сохранения

Формат save v7. Кампании v2–v6 мигрируют автоматически.

## Запуск

```bash
npm install
npm run dev
```

Проверка production-сборки:

```bash
npm run build
```
