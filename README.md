# THE LAST GUILD

Одиночный браузерный процедурный симулятор гильдии исследователей.

## v0.5.1 — Core Loop & UX

Патч связывает мир, людей, экспедиции, бой, политику и развитие гильдии в понятный первый цикл.

### Новый старт

- история гибели прежнего состава;
- старый ветеран, переживший катастрофу;
- цели первых 30 дней;
- центр срочных решений;
- пошаговое обучение через реальные действия.

### Управление временем

- день, неделя и месяц;
- ускорение до следующего важного события;
- автоматическая остановка;
- отображение причины остановки времени.

### Сохранения

- автосохранение;
- несколько ручных слотов;
- экспорт кампании в JSON;
- импорт сохранения;
- резервная копия перед импортом и миграцией;
- восстановление резерва.

### Настройки симуляции

Конкурирующие гильдии можно отключить полностью. При выключении удаляются:

- чужие организации;
- их активные экспедиции;
- гонки за контрактами;
- переманивание сотрудников;
- соперники в великих контрактах.

Политика государств, кризисы, филиалы и поколения продолжают работать.
