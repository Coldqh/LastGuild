# THE LAST GUILD

Одиночный браузерный процедурный симулятор гильдии исследователей.

## v0.7.1 — Interface Rework

Интерфейс переработан после v0.7:

- Штаб снова является только обзором организации;
- Наём, Помещения, Должности, Академия, Совет и Наследие открываются напрямую из боковой панели;
- навигация разделена на группы;
- вторичные блоки управления сворачиваются;
- школы и поколения больше не смешаны с текущим обучением;
- центр решений и карточки стали компактнее;
- сохранение остаётся в формате `save v8`.

## Основные системы

- процедурная генерация мира;
- государства, города, экономика, войны и торговля;
- гильдия, штаб, филиалы и конкуренты;
- персонажи, поколения, академия и наставники;
- экспедиции, подземелья, автобой и бестиарий;
- архив, открытия, последствия и историческая карта;
- несколько слотов, импорт и экспорт сохранений;
- настройки тяжёлых систем и dev-панель.
